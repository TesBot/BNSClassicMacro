# AutoHotKey 脚本使用说明
## 运行方式
1. 安装 AutoHotKey v2.0 版本
2. 以管理员身份运行对应的 `.ahk` 脚本文件；或直接运行编译后的 `.exe` 可执行文件
3. 运行后会在脚本/程序的当前目录自动生成对应的 `.ini` 配置文件

## 配置与功能说明
### 1. 气功-极推龙 使用说明
| 快捷键 | 功能说明 |
|--------|----------|
| CTRL+Q | 取色当前鼠标位置的 X、Y 坐标及对应颜色值，结果显示在屏幕左上角，并缓存该坐标 |
| CTRL+W | 根据已缓存的坐标，重新获取该位置的颜色值，结果显示在屏幕左上角 |
| CTRL+P | 重新读取配置文件数据，并在控制台/界面打印所有配置值 |

### 2. 召唤-功血 使用说明
- 该脚本自带 GUI 操作界面，运行后可查看界面内的使用说明
- 首次运行脚本/程序后，会在当前目录生成配置文件：`zhaohuan_gongxue_config.ini`
